    Resumo:

    Inicialmente, foi necessário criar uma espesificação das classes a serem representadas em Turtle:

    • Prizes
      • year
      • category
      • overallMotivation
      • Laureates
    • Laureate
      • firstname
      • surname
      • motivation
      • share
    Após isto, utilizado Protégé, foi criado o ficheiro TTL com a representação das relações entre as entidades.

    De seguida, foi criado um script Python capaz de converter o ficheiro json em Turtle.

    Finalmente, foi utilizado o comando $ cat para concatenar os ficheiros anteriores. Além disso, foi utilizado o comando $ rapper -c -i turtle prizes.ttl para verificar a estrutura do documento.

    Resultados:

    • Conversor do Ficheiro JSON (prize.json) para Turtle (out.ttl)
    • Ficheiro Json que comtém o dataset
    • Ficheiro Turtle originado pelo Protégé
    • Ficheiro Turtle originado pelo Parser
    • Ficheiro Turtle originado pela concatenação dos 2 Turtle anteriores
