    Resumo:

    Inicialmente, foi necessário criar uma expecificação das class Molecule, para ser representada

    De seguida, foi necessário criar duas Object Properties, que servem para representar as ligações entre moleculas e elementos:

    • hasElement - Representa ligações entre Elementos (Ex:H2)
    • hasMolecule - Representa ligações entre Moleculas (ExH2O)

    De forma a garantir uma correta representação da quantidade presente em cada molecula, foi necessario implementar a Data Property, quantity.

    Por fim foram criadas as várias entidades para representar as moleculas propostas.
